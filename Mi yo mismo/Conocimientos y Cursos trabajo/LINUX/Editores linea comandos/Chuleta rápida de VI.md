## Modos
- ESC → modo comando
- i → insertar
- a → insertar después
- o → nueva línea

## Navegación
- h j k l → mover
- w / b → palabras
- 0 / $ → inicio / fin de línea
- gg / G → inicio / fin del archivo

## Edición
- dd → borrar línea
- yy → copiar línea
- p → pegar
- u → deshacer
- Ctrl+r → rehacer

## Búsqueda
- /texto → abajo
- ?texto → arriba
- n / N → siguiente / anterior

## Guardar / salir
- :w → guardar
- :q → salir
- :wq → guardar y salir
- :q! → salir sin guardar
