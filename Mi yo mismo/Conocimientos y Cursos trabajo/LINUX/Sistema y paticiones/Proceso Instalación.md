
## 1. Particiones e Instalación
- Una partición es una división del espacio físico del disco.
- Mínimo necesario en Linux:
  - Partición para el sistema (`/`).
  - Partición para área de intercambio (swap).
- Se pueden crear particiones adicionales para mejorar rendimiento y seguridad:
  - `/boot`, `/home`, `/var`, `/usr`
