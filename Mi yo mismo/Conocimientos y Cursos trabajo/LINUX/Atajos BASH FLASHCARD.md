
=================

TAB        -> Autocompletar
Ctrl+R    -> Buscar en historial
↑ / ↓     -> Navegar historial
!!        -> Último comando
!$        -> Último argumento

Ctrl+A    -> Inicio línea
Ctrl+E    -> Final línea
Alt+B/F  -> Mover por palabras

Ctrl+U    -> Borrar inicio
Ctrl+K    -> Borrar final
Ctrl+W    -> Borrar palabra

Ctrl+Y    -> Pegar
Ctrl+C    -> Matar proceso
Ctrl+Z    -> Suspender
fg / bg   -> Reanudar

Ctrl+L    -> Limpiar pantalla
Ctrl+D    -> Salir
Alt+.     -> Último argumento
