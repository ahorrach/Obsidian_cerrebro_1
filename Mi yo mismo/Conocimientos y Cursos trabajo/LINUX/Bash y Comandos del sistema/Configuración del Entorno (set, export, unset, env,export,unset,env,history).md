
- **`set`**: Muestra o modifica la configuración del entorno.
    
    - `set -o [opción]`: Activa una opción.
        
    - `set +o [opción]`: Desactiva una opción.
        
- **`export`**: Crea o modifica variables de entorno (ej. `export USERNAME=User`).
    
- **`unset`**: Elimina una variable del entorno.
    
- **`env`**: Muestra variables de entorno o ejecuta comandos con variables modificadas temporalmente.
    
- **`history`**: Muestra el historial de comandos. Al cerrar la sesión, se vuelcan en `~/.bash_history`.
